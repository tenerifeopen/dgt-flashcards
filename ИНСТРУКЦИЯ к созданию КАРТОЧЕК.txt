📌 Как теперь правильно добавлять новые темы
1. Создать файл

В папке:

public/cards/

Например:

new-topic.txt
2. Заполнить файл

Формат строго:

вопрос = ответ

Пример:

¿Qué es una rotonda? = Круговое движение
¿Se puede adelantar aquí? = Можно ли обгонять здесь

❗ Важно:

только один =

без пустых строк

кодировка обычная (как сейчас)

3. Добавить тему в код

В App.jsx → массив topics

Просто добавить строку:

{ name: "Новая тема", file: "/cards/new-topic.txt" }
4. Сохранить и отправить

В твоём .bat (как ты делал):

git add .
git commit -m "add topic"
git push
⚠️ Если тема не появилась

Проверить:

имя файла совпадает (/cards/...)

нет пробелов в названии файла ❗

нажать Ctrl+F5 на сайте
