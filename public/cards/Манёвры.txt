estacionar un turismo = припарковать легковой автомобиль
el conductor se aleja = водитель отлучается (уходит)
dejar engranada la marcha atrás = оставить включенной заднюю передачу
la primera velocidad = первая передача
posición de estacionamiento = положение парковки (режим "P" на автоматической КПП)
ser adelantado = быть обгоняемым (когда тебя обгоняют)
facilitar la maniobra = облегчить маневр
aumentar la velocidad = увеличить (набрать) скорость
encender el indicador de dirección = включить указатель поворота (поворотник)
un desplazamiento lateral = боковое смещение (перестроение из полосы в полосу)
debe señalizarse = должно обозначаться / сигнализироваться
con las luces indicadoras de dirección = с помощью световых указателей поворота
con el brazo = с помощью руки
